Chat → Feed switch on mobile

Four ways to make "go back to the feed" obvious while the chat overlay is open. Every phone starts with the chat open so you can see each option's switch right away. Tap the switch to go to the feed, tap the input pill to bring the chat back. The current production version is the tiny chevron pill, shown last for comparison.
